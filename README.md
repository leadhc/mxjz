# 妙享记账小程序

## 项目介绍

妙享记账是一款基于 uni-app 框架开发的记账小程序，旨在帮助用户轻松管理个人财务，记录日常收支，合理规划预算。
本项目仅提供小程序的开源代码，暂无对应后端服务

![项目Logo](static/logo.png)

## 如果你觉得这个仓库有用，请点一个 Star 支持一下我，谢谢！

### 主要功能

- 📊 **预算管理**：设置月度预算，实时追踪支出进度
- 📝 **快速记账**：支持收入/支出分类，自定义备注和标签
- 📈 **数据统计**：直观展示收支状况，提供月度财务概览
- 🗺️ **位置标记**：支持记录消费位置信息
- 📚 **社区交流**：用户可以浏览、发布记账心得和理财经验
- ❤️ **个性化收藏**：收藏感兴趣的文章和记账模板

## 技术栈

- **前端框架**：Vue 3 + uni-app
- **状态管理**：Pinia
- **构建工具**：Vite
- **样式预处理器**：SCSS
- **代码规范**：ESLint + Prettier
- **地图服务**：高德地图API

## 核心功能详解

### 1. 首页功能

- **本月预算展示**：显示总预算、已使用金额、剩余金额和使用百分比
- **预算进度条**：直观展示预算使用情况，颜色随使用比例变化
- **每日可支出**：根据剩余预算和天数计算每日建议支出
- **本月收支概览**：快速查看收入、支出和结余
- **最近交易记录**：按日期分组展示最近的收支记录

### 2. 记账功能

- **支出/收入切换**：快速切换记账类型
- **金额输入**：支持精确到分的金额输入
- **智能分类**：提供常用收支分类，支持自定义图标
- **日期选择**：默认记录当前日期，支持修改
- **位置标记**：集成地图服务，记录消费位置, 需要申请高德地图key
- **备注功能**：支持添加详细备注信息

### 3. 论坛功能

- **文章浏览**：浏览其他用户分享的记账心得和理财经验
- **文章发布**：分享自己的记账技巧和生活感悟
- **互动功能**：支持点赞、评论、收藏和分享文章
- **搜索功能**：快速查找感兴趣的文章内容

### 4. 个人中心

- **个人信息管理**：修改头像、昵称等基本信息
- **我的收藏**：查看收藏的文章和记账模板
- **我的关注**：管理关注的用户和内容
- **通知设置**：个性化通知偏好设置

## 安装与开发

### 环境要求

- Node.js >= 14.0.0
- npm >= 6.0.0
- HBuilderX（推荐，用于uni-app开发）
- 微信开发者工具（用于小程序预览和调试）

### 安装步骤

1. **克隆项目**

```bash
git clone https://github.com/leadhc/mxjz.git
cd mxjz
```

2. **安装依赖**

```bash
npm install
```

3. **运行项目**

- 使用 HBuilderX 打开项目，选择对应的运行平台（微信小程序、H5等）

4. **导入到微信开发者工具**

- 编译完成后，将生成的 `unpackage/dist/dev/mp-weixin` 或 `unpackage/dist/build/mp-weixin` 目录导入到微信开发者工具中
- 配置AppID（测试环境可以使用测试号）
- 即可预览和调试小程序

## 项目配置

### 高德地图API配置

1. 注册高德地图开发者账号并申请Web服务API Key
2. 在 `pages/account/account.vue` 文件中修改 `mapKey` 变量：

```javascript
const mapKey = ref('your-amap-api-key')
```

### 项目构建设置

- **ESLint配置**：修改 `.eslintrc.js` 文件自定义代码规范
- **Prettier配置**：修改 `.prettierrc` 文件自定义代码格式化规则
- **uni-app配置**：修改 `manifest.json` 和 `pages.json` 文件配置小程序信息和页面路由
